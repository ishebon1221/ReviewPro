# ReviewPro ⭐

نظام ذكي لإدارة تقييمات Google Maps للعيادات الطبية باستخدام الذكاء الاصطناعي.

## ✨ المميزات الرئيسية

- 🔍 **تحليل تقييمات Google Maps** - جلب وتحليل التقييمات تلقائياً
- 🤖 **الذكاء الاصطناعي** - تحليل عميق للتقييمات باستخدام Claude AI
- 💬 **توليد الردود** - إنشاء ردود احترافية على التقييمات تلقائياً
- 🎯 **خطط التحسين** - استراتيجيات 90 يوم لرفع التقييمات
- 📊 **التقارير المفصلة** - تقارير شاملة عن أداء العيادة
- 📱 **تطبيق ويب** - تطبيق PWA يعمل أوفلاين
- 🌙 **واجهة مظلمة** - تصميم عصري وسهل الاستخدام

## 🚀 البدء السريع

### المتطلبات
- متصفح حديث (Chrome, Firefox, Safari, Edge)
- مفاتيح API:
  - **Claude AI API** من [Anthropic](https://console.anthropic.com)
  - **Google Maps API** من [Google Cloud](https://console.cloud.google.com)

### التثبيت

1. **استنساخ المستودع**
```bash
git clone https://github.com/ishebon1221/ReviewPro.git
cd ReviewPro
```

2. **فتح التطبيق**
```bash
# افتح الملف في المتصفح مباشرة
open index.html

# أو استخدم خادم محلي (مُفضّل)
python -m http.server 8000
# ثم اذهب إلى: http://localhost:8000
```

3. **إضافة مفاتيح API**
   - اذهب إلى **الإعدادات** ⚙️
   - أضف **مفتاح Claude API**
   - أضف **مفتاح Google Maps API** (اختياري - لجلب التقييمات)

## 🔑 إعداد مفاتيح API

### Claude AI API

1. اذهب إلى [Anthropic Console](https://console.anthropic.com)
2. سجل حساب جديد أو سجل دخول
3. اذهب إلى **API Keys**
4. انسخ المفتاح وأضفه في الإعدادات
5. اختر نموذج: `claude-haiku-4-5-20251001` (اقتصادي وسريع)

### Google Maps API (اختياري)

1. اذهب إلى [Google Cloud Console](https://console.cloud.google.com)
2. أنشئ مشروع جديد
3. فعّل **Places API** و **Maps JavaScript API**
4. أنشئ **API Key**
5. قيّد المفتاح للـ Web origins فقط
